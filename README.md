# Arithmetic-Logic-Unit-ALU-Design-in-Logisim
This project implements a manually designed n-bit Arithmetic Logic Unit (ALU) in Logisim Evolution without using built-in ALU components.

# Components
- Half Adder
- Full Adder
- Ripple-carry adder
- Arithmetic unit
- B-input logic
- Logical unit
- 2:1 multiplexer
